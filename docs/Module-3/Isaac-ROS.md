
# Isaac ROS

Isaac ROS enables **hardware-accelerated ROS 2 nodes**.

### Capabilities:

- Visual SLAM (VSLAM) for mapping and localization  
- Sensor data processing (camera, LiDAR, IMU)  
- Real-time navigation and control  

### Example:

- Robot reads depth camera input  
- Runs VSLAM to map environment  
- Publishes pose and map data over ROS 2 topics
