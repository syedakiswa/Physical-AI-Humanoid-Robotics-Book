# NVIDIA Isaac Sim

Isaac Sim provides **photorealistic 3D simulation** and **synthetic data** for AI training.

### Features:

- Realistic physics and lighting  
- Synthetic image and sensor data generation  
- Integration with ROS 2 for robot control  
- Supports humanoid robots and multiple sensors

### Benefits:

- Train AI in simulation safely  
- Generate large datasets for vision-based tasks  
- Test perception, grasping, and navigation algorithms
