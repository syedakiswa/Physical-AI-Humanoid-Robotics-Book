# Introduction to NVIDIA Isaac

NVIDIA Isaac is a **robotics AI platform** for simulating, training, and deploying intelligent robots.

### Focus Areas:

- Advanced perception (vision, LiDAR, depth)  
- AI-based navigation and path planning  
- Realistic simulation and synthetic data generation  

### Key Components:

- **Isaac Sim**: Photorealistic simulation environment  
- **Isaac ROS**: ROS 2 nodes with hardware acceleration  
- **Nav2**: Path planning for bipedal humanoids
