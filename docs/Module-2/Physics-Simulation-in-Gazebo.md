# Physics Simulation in Gazebo

Gazebo allows us to simulate **real-world physics**.  

### Key Physics Concepts

1. **Gravity** – How the robot falls or stands  
2. **Collisions** – Detects when robot hits objects  
3. **Friction** – How surfaces affect motion  
4. **Rigid Body Dynamics** – Simulates robot parts moving naturally  

### Robot Components

- **Links**: The physical parts (arms, legs, torso)  
- **Joints**: Connects links (rotational or sliding)  
- **Forces and Torques**: Move robot parts realistically  

> Example: If a robot arm pushes a ball, Gazebo calculates the resulting motion based on forces and collisions.
