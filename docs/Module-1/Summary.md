# Summary of Module 1: TF2 – The Sacred Tree

- TF2 tracks the relationship between robot frames.  
- Robots rely on TF2 for navigation, sensing, and manipulation.  
- The TF2 tree organizes all frames in order.  
- Transforms can be static or dynamic.  
- ROS2 provides tools to publish and visualize transforms.  
- TF2 is essential for *every Physical AI system*.  

By mastering TF2, you unlock **real robotics** — the foundation of humanoids, drones, autonomous vehicles, and more.
