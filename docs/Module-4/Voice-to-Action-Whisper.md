# Voice-to-Action Using OpenAI Whisper

Whisper is a speech-to-text model that converts **voice commands into clean text**.

### Workflow:

1. User speaks: “Clean the room”
2. Whisper converts speech → text
3. Text goes to the LLM for planning

### Features of Whisper:

- Handles different accents
- Understands noisy environments
- Provides accurate transcriptions

### Why Robots Need Voice Commands?

- Hands-free operation
- Natural interaction
- Makes humanoids user-friendly
