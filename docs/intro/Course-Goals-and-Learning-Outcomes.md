# Course Goals

By completing this course, students will:

- Understand the **foundations of Physical AI**  
- Learn how **humanoid robots** work  
- Build simulations using **Gazebo** and **Unity**  
- Control robots using **ROS 2**  
- Use **NVIDIA Isaac** for advanced AI perception  
- Combine **Language + Vision + Robotics**  
- Design a full **Physical AI pipeline**  

---

# Learning Outcomes

Students will gain the ability to:

## 1. Understand Embodied Intelligence
Learn how AI systems behave and make decisions in the **physical world**, dealing with motion, balance, and real-world dynamics.

## 2. Master ROS 2
Work with:
- Nodes  
- Topics  
- Services  
- URDF models  
- Robot control interfaces  

## 3. Simulate Robots
Build and test robots in virtual spaces with:
- Physics  
- Collisions  
- Sensors  
- Realistic environments  

## 4. Use NVIDIA Isaac Sim
Develop advanced robotics features such as:
- Synthetic data generation  
- VSLAM  
- Navigation  
- Path planning  

## 5. Work With Voice, Vision & Language
Use:
- **Whisper** for voice input  
- **GPT-based planners** for reasoning  
- **Computer vision** for perception  

## 6. Build a Complete Humanoid AI System
Create a full pipeline from:  
**Simulation → Control → Perception → Language → Action**
